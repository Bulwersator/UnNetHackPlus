About branches in git repository:
master - main branch, contains current version of UnNetHackPlus
wiki - clone of UNH+ notes hosted on github wiki (see https://github.com/Bulwersator/UnNetHackPlus/wiki/_access for information about wiki git repository)
github - backup of other github metadata, maintained with https://github.com/joeyh/github-backup
new-randomized-appearances - additional randomized appearances, merge to master is currently not planned as this would break saves

Other branches are various features undergoing testing and/or development.
